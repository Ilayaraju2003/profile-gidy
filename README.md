# profile-gidy

# GIDY Technical Task

## Project Overview
This project was developed as part of the GIDY technical assignment.  
It includes a **React frontend** and a **Node.js backend**, with database management using **Prisma**.

The project is deployed online and the source code is publicly available.

Live Demo: https://profile-gidy.vercel.app/                       

---

# Tech Stack

This project uses the following technologies:

### Frontend
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6)
- HTML5 & CSS3

### Backend
- Node.js
- Express.js

### Database
- Prisma ORM

### Development Tools
- Git & GitHub for version control
- ESLint for code linting
- PostCSS for CSS processing

### Deployment (Example)
- Vercel / Netlify (Frontend)
- Render / Railway (Backend)

## How It Works

  -Users can perform actions such as creating, viewing, or updating data through the interface.

  -The frontend sends requests to the backend API.

  -The backend processes the request and interacts with the database using Prisma.

  -The updated data is returned and displayed instantly in the UI.

## Output / Result

  -The application dynamically displays the updated data without reloading the page.

  -Users receive immediate feedback after performing actions.

  -Data is stored and retrieved efficiently from the database.

## Benefits

  -Faster interaction with the system

  -Improved user experience

  -Organized data handling

  -Scalable architecture for future improvements
